# Cognizant Digital Nurture 5.0 -- Java Full Stack Engineer

## Overview

This repository contains all assignments, projects, and hands-on exercises completed as part of the **Cognizant Digital Nurture 5.0** program for the **Java Full Stack Engineer (FSE)** role. The curriculum is structured into two tracks: **Upskilling** (foundational modules) and **Deep Skilling** (advanced, role-specific modules).

## Repository Structure

```
cognizant-digital-nurture-5.0/
  upskilling/          Foundational skill-building modules
    ansi-sql/          ANSI SQL assignments
    bootstrap5/        Bootstrap 5 responsive design exercises
    core-java/         Core Java programming assignments
    css3/              CSS3 styling exercises
    html5/             HTML5 markup assignments
    javascript/        JavaScript programming exercises
  deep-skilling/       Advanced, role-specific modules
    angular/           Angular framework assignments
    engineering-concepts/   Software engineering fundamentals
    git/               Git version control exercises
    junit-mockito-sl4j/     Unit testing (JUnit, Mockito) and logging (SLF4J)
    microservices/     Microservices architecture assignments
    pl-sql/            PL/SQL database programming
    react/             React framework assignments
    spring-data-jpa/   Spring Data JPA persistence exercises
    spring-rest/       Spring REST API development
    springcore-maven/  Spring Core with Maven build tool
```

## Modules

### Upskilling

| Module | Topics Covered |
|--------|----------------|
| ANSI SQL | Relational database queries, joins, subqueries, set operations |
| Bootstrap 5 | Responsive layouts, components, utility classes, grid system |
| Core Java | OOP concepts, collections, streams, exception handling, multithreading |
| CSS3 | Selectors, flexbox, grid, animations, transitions, responsive design |
| HTML5 | Semantic markup, forms, multimedia, accessibility, APIs |
| JavaScript | ES6+ syntax, DOM manipulation, async programming, closures, prototypes |

### Deep Skilling

| Module | Topics Covered |
|--------|----------------|
| Angular | Components, services, routing, forms, HTTP client, RxJS |
| Engineering Concepts | Design patterns, SOLID principles, system design, code quality |
| Git | Branching strategies, rebasing, merge conflict resolution, workflows |
| JUnit / Mockito / SLF4J | Unit testing, mocking, test-driven development, logging best practices |
| Microservices | Service decomposition, RESTful communication, containerization basics |
| PL/SQL | Stored procedures, functions, triggers, cursors, exception handling |
| React | Components, hooks, state management, routing, lifecycle methods |
| Spring Data JPA | Entity mapping, repositories, JPQL, criteria API, transaction management |
| Spring REST | RESTful APIs, request mapping, error handling, HATEOAS, security basics |
| Spring Core / Maven | Dependency injection, inversion of control, bean lifecycle, build automation |

## Purpose

This repository serves as a comprehensive portfolio of the work completed during the Cognizant Digital Nurture 5.0 program. It demonstrates proficiency across the full Java full-stack spectrum -- from frontend technologies (HTML5, CSS3, JavaScript, Bootstrap 5, Angular, React) to backend frameworks (Spring Core, Spring REST, Spring Data JPA, Microservices) and database systems (ANSI SQL, PL/SQL), along with supporting competencies in testing, version control, and software engineering principles.

## Getting Started

Each module directory contains its own set of assignment files. Refer to the individual module folders for specific instructions, source code, and related resources.
