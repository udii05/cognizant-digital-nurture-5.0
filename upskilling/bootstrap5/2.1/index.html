Bootstrap Structure and Files

Exercise 2.1 – Purpose of CSS, JS, and Icons folders

CSS Folder

The CSS folder contains Bootstrap stylesheet files used to style web pages. It provides responsive layouts, colors, typography, spacing, buttons, and other UI components.

Common Files:

  • bootstrap.css
  • bootstrap.min.css
  • bootstrap.css.map

JS Folder

The JS folder contains JavaScript files that add interactive features such as modals, dropdowns, tooltips, carousels, accordions, and navigation components.

Common Files:

  • bootstrap.js
  • bootstrap.min.js
  • bootstrap.bundle.js
  • bootstrap.bundle.min.js

Icons Folder

The Icons folder contains Bootstrap Icons used to enhance the appearance of web pages with graphical symbols such as home, search, user, heart, and shopping cart icons.

Note: Bootstrap Icons are downloaded separately from the main Bootstrap package.
