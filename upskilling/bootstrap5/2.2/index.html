Bootstrap Structure and Files

Exercise 2.1 & 2.2

CSS Folder

The CSS folder contains Bootstrap stylesheet files used for designing responsive layouts, typography, buttons, colors, spacing, and other UI components.

  • bootstrap.css
  • bootstrap.min.css
  • bootstrap.css.map

JS Folder

The JS folder contains JavaScript files that provide interactive components such as modals, dropdowns, accordions, carousels, tooltips, and collapse.

  • bootstrap.js
  • bootstrap.min.js
  • bootstrap.bundle.js
  • bootstrap.bundle.min.js

Icons Folder

The Icons folder contains Bootstrap Icons used to improve the appearance of web pages with graphical symbols.

Note: Bootstrap Icons are downloaded separately from the main Bootstrap package.
